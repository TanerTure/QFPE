# QFPE
This repository contains code for the time-evolution of the Generalized Quantum-Fokker Planck Equation (GQFPE). When theaccompanying paper is published; a link will be placed [here](). A derivation and general introduction to the GQFPE can be found in [Jang, Seogjoo *J. Chem. Phys.* 214102 (2016)](https://pubs.aip.org/aip/jcp/article/144/21/214102/313173)

Script files which generate data for analysis or plotting are made by running scripts which begin with `data_`. Script files which produce figures used in the paper begin with `plot_`. All other `.py` files define helper functions which will form the basis of a package.
